## PowerBI-Projekt: Gestaltung einers interaktiven Dashboards für einen fiktiven Fahrradhersteller

![Dashboard-Ausschnitt](https://github.com/user-attachments/assets/136bf50d-2ff9-4850-80e0-2a0eb8f4dcc9)  

- Datengrundlage: Daten zu Verkäufen, Kunden und Produkten des fiktiven Fahrradherstellers "AdventureWorks" (Daten aus PowerBI-Kurs)
- Bereinigung und Transformation der Datensätze in PowerQuery
- Erstellung eines Datenmodells mit Primär- und Fremdschlüsseln in PowerBI Desktop
- Erstellung einer Vielzahl an einfachen und komplexeren Measures mit DAX
- Visualisierung auf vier Berichtsseiten mit verschiedenen Diagrammen, Drilldown-Menüs, Tooltips etc.
